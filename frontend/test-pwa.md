# Чеклист тестирования PWA

## 1. Проверка манифеста
- [ ] Откройте Chrome DevTools → Application → Manifest
- [ ] Проверьте что все поля заполнены корректно
- [ ] Проверьте что иконки загружаются

## 2. Проверка Service Worker
- [ ] Откройте Chrome DevTools → Application → Service Workers
- [ ] Убедитесь что SW зарегистрирован и активен
- [ ] Проверьте статус: "activated and is running"

## 3. Тестирование offline режима
- [ ] Откройте приложение
- [ ] Перейдите на несколько страниц
- [ ] DevTools → Network → Offline
- [ ] Обновите страницу - должна загрузиться из кэша
- [ ] Попробуйте навигацию - должна работать

## 4. Проверка кэширования
- [ ] DevTools → Application → Cache Storage
- [ ] Проверьте что ресурсы кэшируются
- [ ] Проверьте размер кэша

## 5. Lighthouse audit
- [ ] DevTools → Lighthouse
- [ ] Выберите "Progressive Web App"
- [ ] Запустите аудит
- [ ] Score должен быть > 90

## 6. Установка на мобильном
- [ ] Откройте на Android/iOS
- [ ] Проверьте промпт установки
- [ ] Установите приложение
- [ ] Запустите с главного экрана
- [ ] Проверьте standalone режим

✅ Все компоненты PWA настроены и готовы к тестированию
